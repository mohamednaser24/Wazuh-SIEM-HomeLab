# 🛡️ Wazuh SIEM Home Lab

A complete SIEM Home Lab built using Wazuh, OpenSearch, Filebeat, and Wazuh Dashboard on Kali Linux.

---

## 📖 Project Overview

This project demonstrates how to deploy and configure a complete Wazuh SIEM environment for security monitoring and threat detection.

The lab includes:

- Wazuh Manager
- Wazuh Dashboard
- OpenSearch Indexer
- Filebeat
- Windows Agent
- Custom Detection Rules
- Security Monitoring
